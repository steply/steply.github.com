An open source library for photo apps to easily transfer photos between photo apps.

For Developers

A cool feature of the iPhone SDK is an application’s ability to “link” to a custom URL scheme and to launch the app from another app. Using this library and submitting the app makes it possible to transfer photos between your app and many other photo apps. Whenever a new app adds the library and submits their app, the list of apps in CrossAppPhotoTransfer is automatically updated wirelessly and is instantly accesible amongst all apps using this library.

  • Minimizes your effort to go through the complicated setup process to adapt the use of custom URL scheme and image retrieval process
  • Convenient async callback which won't block your main UI thread when processing large images.
  • Dynamic list of apps, automatically updated. No need to update your library to add new apps into the list
  • Provides an out-of-the-box polished template view components. No need to worry about setting up an interface to show the available apps to export to.
  • Highly documented code structure that lets you customize your own look and feel
  • Flexible for adding other scheme handling functionalities
